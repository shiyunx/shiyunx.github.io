#### Built with online resources, tools, and technologies:
- **Minima template** – jekyll theme
- **Jekyll** – static site generator
- **Liquid** – dynamic content
- **Github Pages** – free hosting and automatic deployment  
- **Markdown** – content writing
- **HTML & CSS** – shadow boxes, layout styling and design tweaks

#### Steps
1. Create a new repository in github
2. Add jekyll structure
- Created files and configured _config.yml
3. Design shadow boxes, filters, and layouts
- Added a profile image and text
- Added shadow card styles using inline CSS in home, projects and posts
- Added filters by categories: Automations, Applications, Data, Uncategorised
4. Deploy to github pages

#### Learning outcome
- I gained the skills to create a website through the use of online resources, tools, and technologies.
